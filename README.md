# FastAPI Notes App
A simple notes application built with FastAPI, HTML, and CSS.
Data is stored in a JSON file for learning purposes.

## Features
- Create notes
- View notes
- Delete notes
- Mark notes as completed

## Tech Stack
- FastAPI
- Jinja2 templates
- HTML / CSS
- JSON storage

## How to run

```bash
pip install fastapi uvicorn python-multipart
uvicorn main:app --reload
